### wp core download
- populates the `/var/www/html` with the standard files and latest source code from WordPress official server

### wp config create
- generates `wp-config.php`

### --allow-root
- running as root is a huge security concern
- allowing root means you know what youre doing

### www-data
- it isn't a "WordPress" user; it's the standard web-service user for Debian.
